# DataLabel.Position property (Excel)

Returns or sets an **[XlDataLabelPosition](Excel.XlDataLabelPosition.md)** value that represents the position of the data label.


## Syntax

_expression_.**Position**

_expression_ A variable that represents a **[DataLabel](excel.datalabel(object).md)** object.
