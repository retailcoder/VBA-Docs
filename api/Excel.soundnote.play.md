# SoundNote.Play method (Excel)

> [!NOTE] 
> This method should not be used. Sound notes have been removed from Microsoft Excel.

Plays the sound note.


## Syntax

_expression_.**Play**

_expression_ A variable that represents a **SoundNote** object.
