# LegendKey.Width property (Excel)

Returns a **Double** value that represents the width, in [points](../language/glossary/vbe-glossary.md#point), of the object.


## Syntax

_expression_.**Width**

_expression_ A variable that represents a **[LegendKey](excel.legendkey(object).md)** object.
