# WorkbookQuery.Formula property (Excel)

The Power Query M formula for the object. Read/write **String**.


## Syntax

_expression_.**Formula**

_expression_ A variable that represents a **[WorkbookQuery](Excel.WorkbookQuery.md)** object.
