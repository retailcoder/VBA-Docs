# PageSetup.Pages property (Excel)

Returns or sets the count or item number of the pages in the **[Pages](excel.pages.md)** collection.


## Syntax

_expression_.**Pages**

_expression_ A variable that represents a **[PageSetup](Excel.PageSetup.md)** object.
