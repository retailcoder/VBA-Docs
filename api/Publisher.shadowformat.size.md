# ShadowFormat.Size property (Publisher)

Returns or sets a **Single** that represents the width of the shadow. Read/write.


## Syntax

_expression_.**Size**

_expression_ A variable that represents a **[ShadowFormat](Publisher.ShadowFormat.md)** object.


## Return value

Single
