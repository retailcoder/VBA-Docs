# Cell.Width property (Publisher)

Returns a **Long** that represents the width (in cells) of a table, range of cells, or page. Read-only.


## Syntax

_expression_.**Width**

_expression_ A variable that represents a **[Cell](Publisher.Cell.md)** object.
