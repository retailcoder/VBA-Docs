# PhoneticGuide.Text property (Publisher)

Returns a **String** that represents the contents of phonetic text. Read-only.


## Syntax

_expression_.**Text**

_expression_ A variable that represents a **[PhoneticGuide](Publisher.PhoneticGuide.md)** object.
