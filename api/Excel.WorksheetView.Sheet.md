# WorksheetView.Sheet property (Excel)

Returns the sheet name for the specified **WorksheetView** object. Read-only.


## Syntax

_expression_.**Sheet**

_expression_ A variable that represents a **[WorksheetView](Excel.WorksheetView.md)** object.
