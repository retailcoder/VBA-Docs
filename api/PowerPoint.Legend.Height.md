# Legend.Height property (PowerPoint)

Returns or sets the height, in [points](../language/glossary/vbe-glossary.md#point), of the object. Read/write  **Double**.


## Syntax

_expression_.**Height**

_expression_ A variable that represents a '[Legend](PowerPoint.Legend.md)' object.
