# Tag.Delete method (Publisher)

Deletes the specified object.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a **[Tag](Publisher.Tag.md)** object.


## Remarks

A run-time error occurs if the specified object does not exist.
