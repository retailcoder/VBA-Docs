# CaptionStyle.Parent property (Publisher)

Returns the parent of the **CaptionStyle** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[CaptionStyle](Publisher.CaptionStyle.md)** object.
