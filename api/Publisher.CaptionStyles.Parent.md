# CaptionStyles.Parent property (Publisher)

Returns the parent of the **CaptionStyles** collection. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[CaptionStyles](Publisher.CaptionStyles.md)** object.
