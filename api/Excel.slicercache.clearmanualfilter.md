# SlicerCache.ClearManualFilter method (Excel)

Clears the manual filter for the slicer cache.


## Syntax

_expression_.**ClearManualFilter**

_expression_ A variable that represents a **[SlicerCache](Excel.SlicerCache.md)** object.


## Return value

**VOID**
