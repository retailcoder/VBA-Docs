# Shape.Type property (Excel)

Returns or sets an **[MsoShapeType](Office.MsoShapeType.md)** value that represents the shape type.


## Syntax

_expression_.**Type**

_expression_ A variable that represents a **[Shape](Excel.Shape.md)** object.
