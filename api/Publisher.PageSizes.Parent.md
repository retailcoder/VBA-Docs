# PageSizes.Parent property (Publisher)

Returns the parent of the **PageSizes** collection. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[PageSizes](Publisher.PageSizes.md)** object.


## Return value

Object
