# SortField.SortOn property (Excel)

Returns or sets the attribute of the cell to sort on. Read/write **[XlSortOn](Excel.XlSortOn.md)**.


## Syntax

_expression_.**SortOn**

_expression_ A variable that represents a **[SortField](Excel.SortField.md)** object.
