# DisplayUnitLabel.Parent property (Excel)

Returns the parent object for the specified object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[DisplayUnitLabel](excel.displayunitlabel(object).md)** object.
