# TextConnection.TextFileSemicolonDelimiter property (Excel)

**True** if the semicolon is the delimiter when you import a text file into a query table, and if the value of the **[TextFileParseType](Excel.textconnection.textfileparsetype.md)** property is **xlDelimited**. The default value is **False**. Read/write **Boolean**.


## Syntax

_expression_.**TextFileSemicolonDelimiter**

_expression_ A variable that represents a **[TextConnection](Excel.TextConnection.md)** object.


## Property value

**BOOL**
