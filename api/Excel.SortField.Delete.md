# SortField.Delete method (Excel)

Removes the specified **SortField** object from the **[SortFields](Excel.SortFields.md)** collection.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a **[SortField](Excel.SortField.md)** object.
